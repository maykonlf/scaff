# scaff
Scaffold cli tool

## Commands

### Golang

#### Create new project
```bash
scaff golang init ./path
```

### Add Use Case
```bash
scaff golang add usecase
```

### Add Controller
```bash
scaff golang add controller
```

### Add Presenter
```bash
scaff golang add presenter
```

### Add Infrastructure component
```bash
scaff golang add service
```
